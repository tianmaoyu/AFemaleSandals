# Moppet's Sandal - VirtualBox 自动化管理工具

## 项目简介

Moppet's Sandal 是一个用于自动化管理 VirtualBox 虚拟机的命令行工具，支持批量克隆、启动、关闭和清理虚拟机。适用于软件测试、并行系统配置测试和教学演示等场景。

## 重构说明

本项目已从原始的 .NET Framework 2.0 重构为现代 .NET 8.0，主要改进包括：

### 架构改进
- **现代化框架**：从 .NET Framework 2.0 升级到 .NET 8.0
- **代码结构优化**：使用清晰的类和方法分离，提高可维护性
- **配置类封装**：使用 `UserConfiguration` 类统一管理配置信息
- **LINQ 应用**：简化集合操作逻辑

### 功能改进
- **交互式配置**：更友好的用户输入体验
- **执行确认**：在执行前显示预估信息并请求确认
- **错误处理**：完善的异常捕获和错误提示
- **日志输出**：清晰的分阶段执行日志

### 代码质量提升
- **命名规范**：使用英文命名，符合 C# 编码规范
- **类型安全**：使用泛型集合替代非泛型集合
- **资源管理**：使用 `using` 语句确保资源正确释放
- **注释完善**：关键逻辑添加中文注释

## 系统要求

- .NET 8.0 SDK 或运行时
- VirtualBox 已安装并配置环境变量
- VBoxManage 命令行工具可用

## 编译和运行

### 编译
```bash
cd /workspace/MoppetSandal/src
dotnet build
```

### 运行
```bash
dotnet run
```

## 使用说明

1. **启动程序**后，按照提示输入以下信息：
   - 源虚拟机名称（可添加多个）
   - VirtualBox 虚拟机存储路径
   - 同时启动的虚拟机数量（建议 2-4）
   - 循环次数
   - 虚拟机启动等待时间（分钟）

2. **确认配置**：程序会显示预估的虚拟机总数、硬盘占用和执行时间

3. **执行操作**：
   - 阶段 1：清理旧的克隆虚拟机
   - 阶段 2：按循环次数克隆和启动虚拟机

## 主要功能

### 虚拟机克隆
- 基于源虚拟机批量创建克隆
- 自动添加日期戳和随机数作为唯一标识
- 支持并发克隆多个虚拟机

### 虚拟机管理
- 自动关闭运行中的克隆虚拟机
- 智能清理策略：保留最近 2 个日期的克隆，删除更早的
- 分批处理避免系统资源过载

### 资源估算
- 预估所需硬盘空间
- 预估总执行时间
- 帮助用户合理规划资源

## 注意事项

1. **数据安全**：操作会删除旧的克隆虚拟机，请确保重要数据已备份
2. **资源限制**：根据物理机性能合理设置并发数量
3. **VirtualBox 配置**：确保 VBoxManage 命令可在命令行直接调用
4. **磁盘空间**：每个克隆虚拟机约占用 6GB 空间，请确保有足够磁盘空间
